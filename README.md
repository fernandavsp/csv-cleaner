# 🧹 CSV Cleaner

Sistema para limpeza, padronização e processamento de arquivos CSV em larga escala.

## 🚀 Sobre o projeto

O **CSV Cleaner** foi desenvolvido para automatizar o tratamento de dados, permitindo a limpeza e padronização de arquivos CSV de forma eficiente e escalável.

Ideal para cenários onde há grande volume de dados e necessidade de consistência.

---

## ⚙️ Funcionalidades

* 📌 Remoção de dados inválidos
* 🔄 Padronização de campos
* 📊 Processamento em lote
* ⚡ Alta performance com Go

---

## 🏗️ Arquitetura

O projeto segue princípios de:

* Clean Code
* Separação de responsabilidades
* Arquitetura modular

(Pode evoluir para arquitetura hexagonal)

---

## 🛠️ Tecnologias

* Golang
* Manipulação de arquivos CSV
* Processamento de dados

---

## ▶️ Como rodar

```bash
git clone https://github.com/fernandavsp/csv-cleaner.git
cd csv-cleaner
go run main.go
```
